 ETmaintain
ETmaintain is a Java-based maintenance management system developed as an academic project to demonstrate object-oriented programming concepts.


Features
- Create and track work orders
- Assign technicians automatically based on skill
- Update work order status: PENDING, ASSIGNED, IN_PROGRESS, COMPLETED
- Dashboard displaying total, pending, completed, and high-priority work orders
- View all work orders in a table
- File persistence using CSV files (workorders.csv and technicians.csv)

CSV Files
- workorders.csv
  Columns: id,issue,facility,status,priority,technicianName

- technicians.csv
  Columns: name, skill
